Luke's Library

Please open the console and use the instructions below to model a library. You can create books and either add or remove them from shelves. You can also create shelves and either add or remove them from the library.

Black Text = do not change!

Red Text = change to whatever you want!

  1. To create a new shelf: var shelfname = new Shelf ("Name");
  2. To create a new book: var bookname = new Book ("Title", "Author");
  3. To add a shelf to the library: library.addShelf(shelfname);
  4. To add a book to a shelf: shelfname.addBook(bookname);
  5. To remove a shelf from the library: library.removeShelf(shelfname);
  6. To remove a book from a shelf: shelfname.removeBook(bookname);
  7. To show all shelves in the library: library.showShelves();
  8. To show all books on a shelf: shelfname.showBooks();
